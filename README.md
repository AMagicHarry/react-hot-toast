## Install

```sh
yarn add react-hot-toast
```

or with npm

```sh
npm install react-hot-toast
```

## Basic usage

```jsx
import toast, { Toaster } from 'react-hot-toast';

const App = () => {
  return (
    <div>
      <Toaster />
    </div>
  );
};

// Show notifications anywhere
toast('Hello World');
```
